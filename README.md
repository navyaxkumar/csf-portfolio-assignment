# Portfolio Website

A simple static portfolio website built using HTML and CSS.

## Features
- Home section
- About section (if you have)
- Projects section
- Contact section
- Responsive design (if you added media queries)

## How to Run
Just open `index.html` in any browser.

## Technology Stack
- HTML
- CSS
- VS Code
- Git & GitHub
